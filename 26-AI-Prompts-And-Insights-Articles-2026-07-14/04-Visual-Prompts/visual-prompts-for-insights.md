# Visual Prompts For Insights And Infographics

Use these prompts for a designer or image-generation tool. Keep visuals clean, B2B, software-platform focused, and source-backed.

## Global Visual Style

```text
Create a clean B2B SaaS infographic style for Logic Unit: modern product engineering company, crisp layout, light background, restrained blue/green/charcoal accents, modular cards, platform diagrams, thin lines, subtle depth, professional enterprise feel. Avoid cartoon characters, exaggerated 3D blobs, generic stock people, and unsupported product screens.
```

## Operational Infrastructure SaaS Visual

```text
Create an infographic titled "Operational Infrastructure SaaS". Show a central software platform connected to workflow, data, visibility, compliance, teams, analytics, and integrations. Use clean enterprise SaaS dashboard motifs, not a generic cloud icon. Include industry nodes for retail, livestock, healthcare, logistics, manufacturing, and facilities.
```

## SaaS Operating Company Visual

```text
Create a diagram showing the difference between one-time software delivery and SaaS operating company model. Left side: project delivery. Right side: build, operate, improve, scale, support, ecosystem growth. Use professional B2B SaaS style and Logic Unit brand feel.
```

## Product Engineering Visual

```text
Create a layered product engineering diagram with architecture, user workflows, data infrastructure, integrations, analytics, mobile/web apps, and product iteration. The design should communicate long-term platform value, not outsourcing.
```

## Industry-Specific SaaS Platforms Visual

```text
Create an industry SaaS platform map. Center: Logic Unit. Surrounding verticals: retail, healthcare, livestock, logistics, manufacturing, facilities. Each vertical connects to products or proof: HULM POS, Hospitello, AnimalCare360, Titan CMMS, Seagold, Smart Salik, KS&EW.
```

## AnimalCare360 Visual

```text
Create an AnimalCare360 workflow infographic: cattle/farm records, health tracking, feed and medicine inventory, animal trading, pet hospital/veterinary clinic, FBR billing/POS, CattlePro, Android app. Use agricultural/livestock software cues without using unrealistic animal illustrations. Keep it professional and SaaS-oriented.
```

## Retail Operations Visual

```text
Create a retail operations platform infographic showing POS, inventory, procurement, CRM, employee management, reporting, FBR integration, multi-location visibility. Use HULM POS as the central product proof.
```

## Logistics Visibility Visual

```text
Create a logistics management platform visual showing shipment tracking, document management, customer self-service portal, notifications, operations dashboard, and reporting. Use Seagold as source-backed case study proof.
```

## Strategic SaaS Partnerships Visual

```text
Create a partnership funnel infographic with three routes: strategic SaaS partnerships, system integrator partnerships, enterprise collaboration. Show how each route connects to Logic Unit product ecosystem, technology backbone, case study proof, and contact inquiry.
```
