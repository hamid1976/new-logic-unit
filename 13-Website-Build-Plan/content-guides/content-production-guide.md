# Content Production Guide

## Source Rule

All copy should stay aligned with the current website source unless new business-approved data is provided.

If information is missing, write:

DATA NOT AVAILABLE IN SOURCE

## Approved Positioning Language

Use:

- product engineering company
- SaaS operating company
- industry-specific SaaS platforms
- software platforms for real industries
- product ecosystem
- industry software platforms
- long-term operating model
- digital infrastructure
- strategic partnerships
- system integrators
- enterprise collaboration

Avoid unless source-approved:

- software agency
- custom software development
- app development
- web development
- ERP implementation
- CRM implementation
- IT outsourcing
- managed IT services

## Page Copy Requirements

Each page must include:

- SEO title
- meta description
- primary keywords
- secondary keywords
- H1
- short hero paragraph
- 3-6 main sections
- CTA section
- internal links
- schema recommendation

## Image And Visual Requirements

Homepage:

- Logic Unit brand visual
- product ecosystem visual
- operating model diagram
- industry coverage visual
- technology/capability visual

About:

- company timeline visual
- operating model visual
- product ecosystem diagram

Industry:

- industry grid
- operational software platform illustration
- no unsupported customer imagery

Partnerships:

- partnership routes diagram
- system integrator / enterprise collaboration flow

Contact:

- clean form layout
- inquiry type visual or side cards

## Technical Content Requirements

Create diagrams for:

- Build-operate-scale model
- Product ecosystem model
- Industry coverage
- Technology backbone
- Partnership routes

Diagram copy should stay source-backed.

## Proof Content To Request From CEO

Ask for:

- leadership names and bios
- company registered location or office address
- platform screenshots
- product milestones
- customer logos, if approved
- case studies, if approved
- partner logos, if approved
- hiring/talent information
- investor/expansion details

Do not invent these.
