# SEO Technical Checklist

## Indexing

- [ ] Remove `noindex, nofollow` from production pages.
- [ ] Set production pages to `index, follow`.
- [ ] Confirm Google Search Console can inspect pages.
- [ ] Submit production sitemap.

## Canonicals

- [ ] Home canonical: `https://www.logic-unit.com/`
- [ ] About canonical: `https://www.logic-unit.com/about/`
- [ ] Industry canonical: `https://www.logic-unit.com/industry/`
- [ ] Partnerships canonical: `https://www.logic-unit.com/partnerships/`
- [ ] Contact canonical: `https://www.logic-unit.com/contact/`
- [ ] Detail pages use self-referencing canonicals.

## Sitemap

- [ ] Sitemap includes production URLs only.
- [ ] Sitemap includes all indexable core pages.
- [ ] Sitemap includes industry detail pages after launch.
- [ ] Sitemap includes partnership detail pages after launch.
- [ ] Sitemap excludes blocked/noindex pages.

## Metadata

- [ ] Each page has unique SEO title.
- [ ] Each page has unique meta description.
- [ ] OG title and description match page purpose.
- [ ] OG URL matches canonical.
- [ ] Twitter metadata uses production image URL.

## Structured Data

- [ ] Organization schema uses production URL.
- [ ] WebSite schema added to homepage.
- [ ] Breadcrumb schema added to nested pages.
- [ ] ContactPage schema added to contact page.
- [ ] AboutPage schema added to about page.
- [ ] WebPage schema added where appropriate.

## Heading Structure

- [ ] One H1 per page.
- [ ] Main sections use H2.
- [ ] Subsections/cards use H3.
- [ ] Footer headings do not pollute page H2 structure.

## Internal Links

- [ ] Home links to industry and partnerships.
- [ ] Industry page links to all industry detail pages.
- [ ] Partnerships page links to all partnership detail pages.
- [ ] All detail pages link back to parent page and contact.
- [ ] Contact links back to partnerships.

## Performance

- [ ] Images have width and height.
- [ ] Images are compressed.
- [ ] Fonts are optimized.
- [ ] No unused heavy scripts.
- [ ] Analytics scripts load after interaction where possible.

## Accessibility

- [ ] All images have useful alt text or empty alt for decorative images.
- [ ] Buttons and links have clear labels.
- [ ] Form fields have labels.
- [ ] Color contrast passes.
- [ ] Keyboard navigation works.

## Launch QA

- [ ] Desktop QA complete.
- [ ] Mobile QA complete.
- [ ] Metadata QA complete.
- [ ] Schema validation complete.
- [ ] Sitemap submitted.
- [ ] Search Console inspection complete.
